# Kléber Barber - Barbearia em São Paulo

Site profissional da barbearia Kléber Barber, localizada na Zona Norte de São Paulo.

## 🌟 Características

- **Design Responsivo**: Funciona perfeitamente em desktop e mobile
- **Sistema de Agendamento**: Agendamento direto via WhatsApp
- **Avaliações em Tempo Real**: Sistema de reviews atualizável
- **Localização Integrada**: Mapa e direções via Google Maps
- **SEO Otimizado**: Estrutura semântica HTML5

## 📁 Estrutura do Projeto

```
├── index.html              # Página inicial
├── barbearia_site.html     # Página de avaliações
├── styles.css              # Estilos CSS
├── reviews-data.json       # Dados das avaliações
└── README.md              # Este arquivo
```

## 🚀 Funcionalidades

### Agendamento
- Formulário intuitivo com validação
- Horários de 30 em 30 minutos (9:00 - 18:00)
- Apenas dias úteis (segunda a sexta)
- Integração direta com WhatsApp

### Avaliações
- Dados atualizáveis em tempo real
- Sistema de estrelas e comentários
- Reviews verificados do Google

### Localização
- Endereço completo: R. Antônio de Couros, 84 - Vila Palmeiras, São Paulo - SP, 02726-000
- Mapa integrado do Google Maps
- Botão direto para rotas

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Design responsivo e moderno
- **JavaScript**: Funcionalidades dinâmicas
- **JSON**: Dados das avaliações

## 📱 Como Usar

1. Abra o `index.html` no navegador
2. Navegue pelas seções usando o menu
3. Para agendar: clique em "Agendar" e preencha o formulário
4. Para ver avaliações: clique em "Avaliações"
5. Para localização: clique em "Localização"

## 🔧 Personalização

### Atualizar Avaliações
Edite o arquivo `reviews-data.json` para adicionar novas avaliações.

### Alterar Horários
Modifique o JavaScript no `index.html` para ajustar horários de funcionamento.

### Mudar Endereço
Atualize a seção de localização no `index.html`.
